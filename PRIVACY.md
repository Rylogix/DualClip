# Privacy Policy for DualClip

Effective date: March 28, 2026

DualClip is a desktop clipping and video editing application. This policy explains what DualClip processes, how it is used, and what users control.

## Information DualClip processes

- Screen recordings, video clips, thumbnails, and edited exports created by the user
- Audio captured from selected devices when audio capture is enabled
- File names, file paths, timestamps, and file metadata required to display and manage clips
- App settings and preferences such as capture options, output folders, startup preferences, and editing settings
- Update metadata when checking GitHub releases for portable builds

## How DualClip uses this information

- Recording and saving clips locally
- Showing clip previews, thumbnails, and metadata in the clip library
- Editing, trimming, cropping, splitting, reordering, and exporting clips
- Saving user preferences locally
- Checking for available updates and showing release notes
- Opening external links only when the user explicitly clicks them

## Local storage

DualClip stores clips, exports, caches, logs, and settings on the user's device. DualClip does not require an account.

## Data sharing

DualClip does not sell personal information and does not upload captured video or audio to DualClip-operated servers.

User content remains on the local device unless the user chooses to move or share it elsewhere.

## Third-party services

DualClip may contact:

- GitHub, to check release metadata and download portable app updates
- Discord, if the user clicks the in-app Discord link

Those services are governed by their own terms and privacy policies.

## User control

Users control what DualClip captures, where clips are stored, whether capture starts automatically with Windows, and which clips are deleted or exported.

## Security

DualClip is designed so captured media and settings remain local to the user's device. Users remain responsible for securing their own device and local files.

## Changes to this policy

This policy may be updated over time. Updated versions should include a revised effective date.
